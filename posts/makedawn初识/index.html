MakeDown初识

标题的设置

  1. 使用“#”进行标题设置，“#+空格”表示一级标题，“##+空格”表示二级标题…以此类推；
  2. 在标题下方画横短线，“===”表示一级标题，“—”表示二级标题。

段落

  1. 使用空行来分行
  2. 斜体*txt*，粗体**txt**，下划线<u>txt</u>

区块

段落区块使用>

使用>>可以嵌套

代码区块，若是在段落中可以用反引号将代码块包含，整段的代码可以使用4个空格或者一个制表符（Tab键）或是三个反引号输入语言类型，例如

program main
implicit none
int x
x=1
write(*,*)
end program

e=>end: 结束框
