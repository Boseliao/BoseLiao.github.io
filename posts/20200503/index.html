标题的设置

1.使用使用“#”进行标题设置，“#+空格”表示一级标题，“##+空格”表示二级标题…以此类推；

2.在标题下方画横短线，“===”表示一级标题，“—”表示二级标题。

段落

1.使用空行来分行

2.斜体1，粗体2，下划线3

区块

1.段落区块使用符号>,类似的可以嵌套>>表示二级区块…

2.代码区块，若是在段落中可以用反引号，例如prinf()函数4;整段的代码可以使用4个空格或者一个制表符（Tab键）或是三个反引号输入语言类型，例如

program main
implicit none
int x
x=1
write(*,*)
end program
